# Atlas Agent Startup Procedure

Every agent must follow this procedure before performing any work.

## Required Reading Order

1. docs/v2/project-vision.md
2. docs/v2/requirements-v2.0.md
3. docs/v2/system-architecture-v2.0.md
4. docs/v2/glossary.md
5. docs/v2/decision-log.md
6. Current milestone document
7. agents/workflow.md
8. Agent-specific definition

## Rules

Agents must not:

* Skip required reading
* Invent requirements
* Change architecture
* Expand milestone scope

When uncertainty exists:

Stop work and request clarification.

Project documentation is the source of truth.

Agent assumptions are never considered a source of truth.
