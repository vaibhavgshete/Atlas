# Atlas Project Management Contract v0.1

## Status

Approved for Milestone 1

---

# Purpose

Define the Milestone 1 `Project` entity shape and backend API contract required to support project creation, listing, retrieval, and deletion.

This document is limited to Milestone 1 scope and should be used by the Developer Agent as the implementation contract for project management features.

---

# Scope

This contract covers:

* `Project` entity definition
* Minimum field validation for Milestone 1
* Project management API endpoints
* Request and response shapes

This contract does not cover:

* Repository cloning
* Repository scanning
* Scan entities
* Authentication
* Authorization
* Update project functionality

---

# Project Entity

Milestone 1 introduces a single persistent entity: `Project`.

## Fields

### id

* Type: string
* Required: yes
* Description: unique project identifier generated by Atlas

### name

* Type: string
* Required: yes
* Description: user-provided project name

### repositoryUrl

* Type: string
* Required: yes
* Description: user-provided repository URL associated with the project

### description

* Type: string or null
* Required: no
* Description: optional project description provided by the user

### createdAt

* Type: string
* Required: yes
* Description: timestamp indicating when the project was created

### updatedAt

* Type: string
* Required: yes
* Description: timestamp indicating when the project was last updated

---

# Minimum Validation

Only the minimum validation explicitly supported by the approved milestone and requirements documents is defined for Milestone 1.

## Create Project Input

* `name` must be present
* `repositoryUrl` must be present
* `description` is optional

## Validation Notes

* Milestone 1 does not require provider-specific repository URL validation
* Milestone 1 does not require repository reachability checks
* Milestone 1 does not require duplicate project detection

---

# API Style

Milestone 1 uses a simple JSON HTTP API exposed by the backend.

Base path:

* `/api/projects`

Content type:

* `application/json`

---

# Endpoints

## Create Project

### Request

* Method: `POST`
* Path: `/api/projects`

Request body:

```json
{
  "name": "Atlas Backend",
  "repositoryUrl": "https://github.com/example/atlas-backend",
  "description": "Core service repository"
}
```

### Success Response

* Status: `201 Created`

Response body:

```json
{
  "id": "project_123",
  "name": "Atlas Backend",
  "repositoryUrl": "https://github.com/example/atlas-backend",
  "description": "Core service repository",
  "createdAt": "2026-06-21T10:00:00Z",
  "updatedAt": "2026-06-21T10:00:00Z"
}
```

### Error Response

* Status: `400 Bad Request` when required fields are missing

Response body:

```json
{
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Required fields are missing."
  }
}
```

---

## List Projects

### Request

* Method: `GET`
* Path: `/api/projects`

### Success Response

* Status: `200 OK`

Response body:

```json
[
  {
    "id": "project_123",
    "name": "Atlas Backend",
    "repositoryUrl": "https://github.com/example/atlas-backend",
    "description": "Core service repository",
    "createdAt": "2026-06-21T10:00:00Z",
    "updatedAt": "2026-06-21T10:00:00Z"
  }
]
```

---

## Get Project

### Request

* Method: `GET`
* Path: `/api/projects/{projectId}`

### Success Response

* Status: `200 OK`

Response body:

```json
{
  "id": "project_123",
  "name": "Atlas Backend",
  "repositoryUrl": "https://github.com/example/atlas-backend",
  "description": "Core service repository",
  "createdAt": "2026-06-21T10:00:00Z",
  "updatedAt": "2026-06-21T10:00:00Z"
}
```

### Error Response

* Status: `404 Not Found` when the project does not exist

Response body:

```json
{
  "error": {
    "code": "PROJECT_NOT_FOUND",
    "message": "Project not found."
  }
}
```

---

## Delete Project

### Request

* Method: `DELETE`
* Path: `/api/projects/{projectId}`

### Success Response

* Status: `204 No Content`

### Error Response

* Status: `404 Not Found` when the project does not exist

Response body:

```json
{
  "error": {
    "code": "PROJECT_NOT_FOUND",
    "message": "Project not found."
  }
}
```
